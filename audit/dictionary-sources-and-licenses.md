# 辞書データの出典・ライセンス・照合方針

作成: 2026-07-15 / 監査対象: WordQuest 単語データ（9言語）

## 1. 本監査における外部辞書照合の実態（正直な申告）

本監査は、**専門家（言語学的知識）によるレビュー**として実施した。
本作業環境では、Wiktionary / CC-CEDICT / WordNet / Tatoeba 等の外部辞書へ
**全2万件を1語ずつ、機械的かつ確実に照合する手段が存在しない**。

したがって本監査では、貴殿のルール4〜7に従い、以下を厳守した：

- 実際に照合していない外部辞書へのアクセスを**捏造しない**。
- 各監査記録の `externalSourcesChecked` は原則 **`[]`（空）** とする。
- 外部の権威ある一次資料がなければ確定できない項目（特に発音の強勢位置、
  性・格・活用などの語形細部）は、確信が持てない場合 **`not_verified`** とし、
  `pass` として扱わない。
- 存在しない出典名・URL・引用は**一切作らない**。

外部辞書との突き合わせを厳密に行う場合は、後述の「今後の外部照合手順」に沿って
別途、人間またはオンライン照合環境で実施する必要がある。

## 3. ライセンス上の注意（ルール11遵守）

- es/de/fr の元データは **CC-BY 4.0 / CC-BY-SA** 由来。照合目的の参照と、
  アプリへの取り込み（既に取り込み済み）は区別して扱う。CC-BY-SA由来の文章・例文を
  そのまま大量転載する行為は行っていない（和訳は独自付与・是正済み）。
- 外部辞書の例文・定義文を**無断で大量転載しない**方針を維持。

## 4. 重要な既知の齟齬（監査で判明・要人間確認）

- **ポルトガル語**: 本データは「広く通用する標準（ブラジル/ポルトガル双方で通じる語）」を
  基準にキュレーションされている。貴殿の要件「**ブラジル基準**」とは厳密には一致しない
  可能性がある（欧州寄りの語形・語彙が混在しうる）。→ 監査で `warning`/`requiresHumanReview`
  として扱う。
- **es/de/fr/pt/pl/ru（ピボット言語）**: これらは `answer=英語`, `glosses.ja=和訳`,
  `tags=["word"]` の構造で、**品詞情報・例文・（es/de/fr は）発音を保持しない**。
  したがって「品詞」「例文」「例文訳」「発音」「活用」等の検査は、これらの言語では
  `not_applicable`（該当フィールドなし）となる。
