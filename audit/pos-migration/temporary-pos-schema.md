# 暫定 品詞スキーマ（第2段階）

作成: 2026-07-16 / 大規模データモデル変更を避け、既存 `tags` 配列へ品詞を追加する暫定仕様。

## 許可タグ（日本語ラベル維持）
品詞: 名詞/動詞/形容詞/**副詞/代名詞/前置詞/接続詞/間投詞/限定詞/冠詞/助動詞/数詞/助詞/固有名詞/その他**
特殊: `phrase`（英語文字列・コードが参照・**維持**）
テーマ: phrase問題の tags[1]（**削除・変換しない**）

## 決定
- 単一/複数: 原則単一タグ。同綴りで複数品詞が明確かつ現レコードが複数義を教える場合のみ複数可。
- 並び順: 品詞タグ先頭、phrase/テーマは既存順維持。
- 重複: 禁止（除去）。
- 後方互換: 品詞タグ値はコード非参照ゆえ影響なし。phrase/テーマ維持で機能不変。
- ロールバック: tags のみ変更・Git 差分で復元可能。

## 今回の適用範囲
- スキーマ不足303（英語）と未解決critical447のうち、**answer・辞書品詞・(あれば)例文が一致する高確度・単一品詞のみ** tags を修正。
- 多品詞語・辞書体系差・曖昧は人間確認へ。
